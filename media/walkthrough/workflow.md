## Core workflow

- **Ask → streaming answer**: ask in a board's input box; the answer streams into the node in real time.
- **⑂ Branch**: branch a child conversation off any board without affecting the original (forming a natural DAG).
- **Merge**: box-select multiple boards and start a new conversation from their **deduped, merged** context — this is Braid's moat.
- **Collapsed summaries**: a collapsed board shows an auto-generated structured summary; zooming out switches to an even shorter mini summary.
- **🗜 Compact / context usage**: each board's badge shows the chain's context-window usage %, and you can compact it into a compact node in one click when needed.
- **⌨ Delete**: select a board and press `Delete` to remove it.

> You can reopen this walkthrough anytime from the Command Palette: **Braid: Open Getting Started**.
